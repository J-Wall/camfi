NAME
    camfi apply_edge_thresh - Checks if polyline annotations are close to
    the edge of an image, and if so, convers them to circle annotations by
    computing the smallest enclosing circle of all points in the polyline.

SYNOPSIS
    camfi apply_edge_thresh <flags>

DESCRIPTION
    Checks if polyline annotations are close to the edge of an image, and
    if so, convers them to circle annotations by computing the smallest
    enclosing circle of all points in the polyline.

FLAGS
    --edge_thresh=EDGE_THRESH
        Type: int
        Default: 20
        Minimum distance an annotation has to be from the edge of the
        image before it is converted from polyline to circle

NAME
    camfi add_metadata - Adds image (EXIF) metadata to VIA project by
    reading image files. Optionally spawns multiple processes (reading the
    images is usually I/O bound and can take some time).

SYNOPSIS
    camfi add_metadata [EXIF_TAGS]...

DESCRIPTION
    Adds image (EXIF) metadata to VIA project by reading image files.
    Optionally spawns multiple processes (reading the images is usually
    I/O bound and can take some time).

POSITIONAL ARGUMENTS
    EXIF_TAGS
        Type: str
